to install beyondcode/laravel-websockets
the more simple way is:

add these properties under require in composer.json

"require": {
"guzzlehttp/guzzle": "^7.0.1",
"beyondcode/laravel-websockets": "^1.12",
"pusher/pusher-php-server": "^5.0",
}

run composer update
